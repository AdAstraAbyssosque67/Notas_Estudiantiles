# Actividad 2: Flujo de Control Part. 2 - Registro de Calificaciones

Este repositorio contiene la solución a la Actividad 2 de la asignatura Programación Básica, impartida por el Prof. Gamalier Reyes del Carmen.

## Descripción del Proyecto
El programa es una aplicación de consola desarrollada en C# que permite gestionar las calificaciones de una cantidad n de estudiantes. Utiliza estructuras de control de flujo (bucles for) y arreglos para procesar datos de manera eficiente.

### Funcionalidades principales:
- Entrada dinámica: El usuario define cuántos estudiantes desea registrar.
- Cálculo Automático: Procesa 4 notas por estudiante y calcula el promedio aritmético.
- Lógica de Estatus: Determina automáticamente si el alumno está Aprobado (Promedio ≥ 70) o Reprobado (Promedio < 70).
- Salida Formateada: Muestra los resultados finales en una tabla organizada mediante tabulaciones, facilitando la lectura de los datos.
